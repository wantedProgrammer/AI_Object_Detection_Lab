# YOLOv8 Object Detection and Tracking Lab
A practical implementation of computer vision using YOLOv8 for object detection and tracking across images, video, and live webcam feed, developed as part of the CMPG 313 Artificial Intelligence module.

# 🚀 Key Features

Image Detection: Detects and labels objects in static images with confidence scores

Video Tracking: Tracks objects across video frames with persistent IDs

Live Webcam Detection: Real-time object detection using webcam feed

Automated Annotation: Generates output files with bounding boxes and class labels

Multiple Input Sources: Supports images, video files, and live camera streams

# 🛠 Tech Stack

Model: YOLOv8 (Ultralytics)

Language: Python

Libraries: OpenCV, Ultralytics

Tools: Command Line, Git & GitHub

Environment: Python 3.8+
